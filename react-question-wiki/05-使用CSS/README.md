# 使用 CSS

## 背景

入门前端先学 HTML ，再学 CSS 。<br>
学 React 也是，先学 JSX ，再学 CSS 样式。不过 React 中的 CSS 和 HTML 中的有一些区别。

## 目标

- 学会 React CSS 的几种形式
- 使用 CSS 完善 List 页面样式

## 内容

- 普通方式
    - 内联 style
    - 引入 CSS + className （ `clsx` 或 `classnames` 条件判断 ）
- CSS Module 和 sass
- css-in-js
    - Styled-components
    - Styled-jsx
- 【实战】为 List 和 QuestionCard 增加样式
