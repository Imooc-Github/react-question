# JSX 和 组件

## 背景

学习，要从最基础的开始。

学习前端要从 HTML 开始，学习 React 要从 JSX 和组件开始。

- 组件，就像 HTML 文件

## 目标

- 学会 JSX 基本语法
- 学会 React 组件基础知识
- 使用 JSX 和组件，开发列表页

## 内容

- JSX 语法
- 组件和 props
- 实战：开发列表页
- 加餐：对比 Vue template
- 加餐：JSX 原理
