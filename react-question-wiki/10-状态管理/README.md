# 状态管理

## 背景

使用 Vue 和 React 做项目，只要项目稍微复杂一点，都需要状态管理工具。例如 Vuex redux mobx 等。<br>
状态管理入门比较麻烦，所以先脱离项目环境，先学会这些工具的用法。

## 目标

学会状态管理 Redux 的用法，为项目使用做准备

## 内容

- 为何使用状态管理
- Context
- useReducer
- Redux
- 实战：Redux 开发 todo-list
- Mobx
- 实战：Mobx 开发 todo-list
- 实战：Redux 管理用户信息

## 注意事项

- 现在可能不知道“状态管理”的意义，继续学，等开发完编辑器，就能深刻体会
- 状态管理对于问卷编辑器页面非常重要
